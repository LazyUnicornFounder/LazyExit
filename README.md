## Features

- **Verified MRR and revenue data**
- **Autonomy scoring system**
- **Instant business transfers**
- **Due diligence tools**
- **Seller verification**

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions, Storage)
- **AI:** Claude / Gemini via API
- **Hosting:** Lovable
- **Build Tool:** Vite

## Getting Started

```bash
# Clone the repo
git clone https://github.com/LazyUnicornFounder/lazy-exit.git
cd lazy-exit

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Supabase credentials to .env

# Start the dev server
npm run dev
```
